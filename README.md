Project is created with:

- HTML, CSS, TypeScript
- Helpful development libraries: eslint, webpack and babel

## Setup

To run this project, lint, install libraries, or build it locally using npm:

```
$ npm install
$ npm start
$ npm build
$ npm run lint
$ npm run lint:fix
```
