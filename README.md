## General info

A simple to do app, to explore reactjs with TypeScript.

## Technologies

Project is created with:

* ReactJS 16.8
* HTML, CSS, TypeScript

## Setup

To run this project, lint, install libraries, or build it locally using npm \(node package manager\):

```text
$ npm install
$ npm start
$ npm build
$ npm run lint
$ npm run lint:fix
```
